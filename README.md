# 小深 (XiaoShen) - 计算机新生专属 AI 助教

这是一个基于 DeepSeek-V4 开发的智能助教脚本，专为解决计算机专业大一新生的学习痛点而设计。

## ✨ 特色功能
- **动态画像感知**：根据提问语气自动识别你是“零基础新生”还是“技术大牛”，动态调整讲解深度。
- **自我反思机制**：AI 在回答后会进行“复盘”，自动补充可能遗漏的知识点或优化代码格式。
- **联网搜索增强**：集成 SerpApi，能够查询最新的技术文档和学术资料。
- **精致排版**：严格控制代码缩进，并对数学公式进行终端友好化处理。
- **成本透明**：实时统计每一轮对话的花费，支持设置单次会话预算。
